เชื่อมต่อ Firebase Firestore

สร้าง Firebase Project เปิดใช้ Firestore และเชื่อมต่อกับ Ionic App

เพิ่มข้อมูล (Create)

หน้าเพิ่มรายการรายรับ–รายจ่าย และข้อมูลถูกบันทึกลง Firestore

แสดงข้อมูลแบบ Realtime (Read)

ดึงข้อมูลจาก Firestore แบบ Realtime พร้อมสรุปรายรับและรายจ่าย

แก้ไขข้อมูล (Update)

สามารถแก้ไขข้อมูลรายรับ–รายจ่าย และอัปเดตลง Firestore

ลบข้อมูล (Delete)

มีการยืนยันก่อนลบข้อมูลเพื่อป้องกันความผิดพลาด
